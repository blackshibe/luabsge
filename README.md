# Luabsge

Small "game engine" that integrates Lua with OpenGL.<br/>

![screenshot](luabsge.png)

## dependencies

-   glm
-   freetype2
-   imgui
-   glfw

## compilation

run `./run.sh test` to run the test project

note: LuaBSGE is only tested on Linux right now
